Operational Constraints

✓

Human Proximity Zones

Dynamic speed reduction: Within 3m of detected human - reduce to 0.5 m/s. Within 1.5m - reduce to 0.2 m/s. Within 0.5m - full stop.

✓

Restricted Zones

Geofenced areas prohibit AMR entry: Break rooms, offices, maintenance zones, loading docks during active loading.

✓

Traffic Management

Fleet management system prevents multiple AMRs in narrow aisles. Priority routing for human traffic during shift changes.

✓

Payload Monitoring

Continuous weight sensing. Operations suspended if payload exceeds 500kg or is improperly balanced.

⚠️

ODD Boundary Monitoring

The AMR continuously monitors all ODD parameters. If any parameter exceeds defined thresholds, the system automatically transitions to a safe state (reduced speed or controlled stop) and alerts remote operators.

Navigation Safety Systems

Multi-layer safety architecture for autonomous movement

Sensor Fusion Architecture

✓

Primary LiDAR System (360° Coverage)

Dual-layer 2D LiDAR with 25m range, 0.25° resolution. Detects obstacles, people, and infrastructure changes in real-time. Redundant sensor configuration with automatic failover.

✓

3D Depth Cameras (Front/Rear)

Stereo vision with 10m depth perception. Detects low-height obstacles (50mm+) and overhead clearances. Human pose estimation for predictive collision avoidance.

✓

Ultrasonic Proximity Sensors

16 sensors providing 0-2m close-range detection. Activates protective stop for sudden obstacle appearance within 0.5m safety zone.

✓

IMU & Odometry System

Inertial measurement unit provides tilt detection, velocity monitoring, and dead-reckoning backup for navigation continuity.

Safety-Rated Functions

✓

Safety-Rated SLAM (Simultaneous Localization and Mapping)

Performance Level d (PLd) certified localization system. Continuous self-validation with position uncertainty monitoring. Automatic safe stop if localization confidence falls below threshold.

✓

Dynamic Path Planning with Safety Zones

Real-time path re-planning maintaining 1.5m minimum clearance from humans. Multi-hypothesis trajectory evaluation for safest route selection.

✓

Protective Field Monitoring

Three-zone safety field: Warning zone (3m) - speed reduction, Protective zone (1.5m) - slow speed, Safety zone (0.5m) - immediate stop.

✓

Safe Speed Monitoring

Redundant speed measurement with independent verification. Automatic braking if speed exceeds contextual limits (1.5 m/s open areas, 0.5 m/s near humans).
